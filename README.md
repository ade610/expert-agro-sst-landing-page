# Expert Agro/SST Landing Page

Landing page profissional da **Expert Agro/SST – Engenharia de Segurança do Trabalho** para divulgar e vender treinamentos, cursos, e-books e soluções técnicas.

## Como executar

Como é um projeto estático, basta abrir o arquivo `index.html` no navegador.

Se quiser rodar com servidor local:

```bash
python3 -m http.server 8080
```

Depois acesse: `http://localhost:8080`

## Arquivos

- `index.html`: estrutura da landing page.
- `styles.css`: estilos responsivos e identidade visual.
